\# <<Arcane Dungeon>>



\## 게임 소개



\- 장르: 1인칭 3D 로그라이크 핵앤슬래시

\- 플레이 방식: 싱글플레이 (랭킹 등록은 온라인으로만 가능)

\- 개발 기간: 2025.10 - 2025.11

\- 팀: 국민대학교 2025년 2학기 게임소프트웨어 팀5

\- 팀원 및 역할 : 조순규, 이정윤, 전경진, 박현빈, 윤신지

무작위로 생성되는 던전에서 적들을 물리치고 강해지세요.
충분히 강해졌다면 보스를 찾아가 처치하세요.
높은 난이도로 최대한 빨리 클리어하여 랭킹을 등록하세요.

\## 조작

\- 입력 장치 : 키보드, 마우스
\- 이동 : WASD + 쉬프트(달리기)
\- 기본 공격 : 좌클릭
\- 점프 : 스페이스
\- 습득한 스킬 상태 확인 : TAB
\- 지도 펼치기 : C
\- 설정 열기 : ESC
\- 스킬 획득 : 마우스 좌클릭
